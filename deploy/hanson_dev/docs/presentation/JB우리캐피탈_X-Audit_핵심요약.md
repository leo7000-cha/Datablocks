## 2. X-Audit — 접속기록 + 이상행위 탐지 + 소명

### 5W1H 접속기록 자동 수집

| 항목 | 내용 |
|------|------|
| **Who** | 접속자 계정 |
| **When** | 접속 일시 |
| **Where** | 접속 IP / 단말 |
| **What** | 작업 유형 (조회/수정/삭제/다운로드) |
| **Whom** | 대상 테이블 (PII 등급 자동 분류) |
| **How** | 접근 경로 (DB Audit / WAS / Agent) |

### 3중 수집 채널

| 방식 | 역할 |
|------|------|
| **DB Audit Log** | S/A급 PII 테이블 선별 수집, SQL 원본 확보 |
| **WAS Log** | 사용자/화면/업무 컨텍스트 (Who/Why 보강) |
| **Java Agent (PSM)** | JDBC 레벨 실시간 가로채기, 코드 수정 불필요 |

> 리스크 기반 선별 로깅 → **연산 60% 절감, 저장 80% 절감**

## 3. 이상행위 자동 탐지 — 7가지 규칙

| 코드 | 규칙 | 조건 | 심각도 |
|------|------|------|--------|
| R01 | 대량 조회/다운로드 | 시간 내 N건 이상 | **HIGH** |
| R02 | 야간/휴일 접근 | 22:00~06:00 | MEDIUM |
| R03 | 접근 거부 반복 | 시간 내 N회 | **HIGH** |
| R04 | 고등급 PII 대량 접근 | 1급 개인정보 N건 | **HIGH** |
| R05 | 동일 테이블 반복 | 시간 내 동일 테이블 N회 | MEDIUM |
| R06 | 미등록 IP 접속 | 90일 미사용 IP | MEDIUM |
| R07 | 장기미사용 계정 | 90일+ 미접속 재접근 | **HIGH** |

> 규칙 추가/수정 가능 — 금융권 특성에 맞게 유연 설정

## 4. 소명 워크플로우 — 탐지부터 감사 증빙까지 자동화

```
이상행위 자동 탐지
    ↓
소명 요청 자동 발송 (이메일, 72시간 기한)
    ↓
담당자 소명 작성 (로그인 불필요, 링크 클릭)
    ↓
관리자 검토 → 승인 / 반려(재소명)
    ↓
SLA 미응답 시 자동 에스컬레이션
    ↓
월간점검 보고서에 자동 포함 → 감사 대응 완결
```

### 소명 프로세스 핵심 특징

- **자동화**: 탐지 → 통보 → 소명 → 승인 전 과정 API 기반 자동
- **기한 관리**: 48시간 SLA, 미응답 시 자동 OVERDUE → ESCALATED
- **공개 소명 페이지**: 대상자가 로그인 없이 링크만으로 소명 제출
- **감사 추적**: 모든 처리 이력 기록, 보고서에 자동 반영

## 5. 위변조 방지 — SHA-256 해시 체인

```
레코드1: hash(데이터 + "GENESIS")
레코드2: hash(데이터 + 레코드1_해시)
레코드3: hash(데이터 + 레코드2_해시)
   ...
→ 중간 1건 변조 시 이후 모든 해시 불일치 → 즉시 탐지
```

- 매월 자동 검증 (안전성확보조치 제8조: 월 1회 이상)
- 수동 검증도 언제든 가능
- 검증 결과 VALID/INVALID 즉시 표시

## 8. ONE PLATFORM 핵심 장점

### X-Scan이 발견한 PII가 전 모듈에 즉시 공유

```
X-Scan (PII 탐지)
    ↓ PII 메타데이터 자동 공유
    ├→ X-Purge: "이 테이블에 S급 PII → 파기 대상 자동 포함"
    ├→ X-Audit: "이 테이블 접근 → S급 PII 접근으로 자동 분류 → 즉시 알림"
    └→ X-Gen:   "이 테이블 복사 → 주민번호 자동 마스킹"
```

### 개별 도입 vs X-One

| 구분 | X-One | 개별 솔루션 |
|------|-------|------------|
| PII 메타 | **1회 스캔 → 전 모듈 공유** | 솔루션마다 개별 등록 |
| 인프라 | **단일 서버/DB** | 솔루션별 별도 (3~4배) |
| 관리 콘솔 | **통합 1개** | 솔루션별 별도 |
| 감사 대응 | **"X-One 도입" 1건** | 영역별 개별 증빙 |
| 비용 | **통합 라이선스 1건** | 개별 × 3~4 |

### JB우리캐피탈 특수 이점

> **X-Gen이 이미 도입 완료** → KeyMap(연관관계) + DB환경 확보
> → X-Purge + X-Audit 약 **6주** 만에 거버넌스 완성 (일반 대비 60% 단축)
